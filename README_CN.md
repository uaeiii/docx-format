# docx-format — Claude Code 技能：福州大学毕业论文格式规范

一个 **Claude Code** 技能，用于按照 **《福州大学本科生毕业设计（论文）撰写规范》（福大教〔2019〕37号）** 自动生成符合学校规范的 Word 文档。

## 功能特性

- **页面设置** — A4 纸，精确边距（上25mm，下20mm，左25mm，右20mm）
- **字体映射** — 所有标题级别使用正确的字体和字号（黑体/宋体）
- **标题层次** — 同时支持理工类（第1章, 1.1, 1.1.1）和文法经管类（一、（一）、1.）标题样式
- **行距控制** — 正文固定行距20pt，参考文献行距17pt
- **页眉页脚** — 奇偶页不同页眉（论文题目/福州大学本科生毕业设计（论文）），页码居中
- **表格规范** — 无左右边线，表名在上方居中
- **图题规范** — 图名在下方居中，按章编号（图2-5）
- **公式规范** — 公式居中，编号右对齐
- **参考文献** — GB/T 7714-2015 标准格式，含各类文献类型示例
- **分章控制** — 每章自动分页

## 工作流程

本 skill 生成论文文档时遵循以下步骤：

1. **收集信息** — 确认论文类型（工程设计/实验研究/计算机软件/经管文科），收集题目、作者、章节内容和参考文献
2. **配置页面** — 设置 A4 纸、边距、奇偶页页眉和页码
3. **构建章节** — 生成封面 → 中文摘要 → 英文摘要 → 目录 → 正文 → 参考文献 → 附录 → 致谢
4. **应用格式** — 应用规范中所有字体、字号、间距和布局规则
5. **格式验证** — 逐项核对格式要求验证清单

```bash

git clone https://github.com/uaeiii/docx-format.git

# 复制到 Claude Code 的 skills 目录
cp -r docx-format ~/.claude/skills/
```

## 规范依据

本 skill 严格依据《福州大学本科生毕业设计（论文）撰写规范》（福大教〔2019〕37号）执行。

核心格式要求：
- 纸张：A4 (210×297mm)
- 页边距：上25mm，下20mm，左25mm，右20mm
- 正文：宋体小四号（12pt），固定行距20pt
- 章标题：黑体小二号（18pt），段后30-36pt，居中
- 一级节标题：黑体三号（16pt），段后18-24pt
- 二级节标题：黑体小三号（15pt），段后12-15pt
- 三级节标题：黑体小四号（12pt），段后6-9pt
- 参考文献：GB/T 7714-2015 格式
